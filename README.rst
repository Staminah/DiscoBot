DiscoBot
========

A Simple Musical Bot for Discord. (Work in progress)

Getting Started
---------------

Requirements
~~~~~~~~~~~~

::

    $ pip install -r requirements.txt

Installing
~~~~~~~~~~

1. Add **ffmpeg.exe** to your PATH
2. Replace “token” by your bot token in **disco-bot.py** (at the very
   bottom)

::

    client.run('your_token_here')

Running
-------

1. Run disco-bot.py

::

    python disco-bot.py
